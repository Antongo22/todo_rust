# todo_rust

Консольный todo-лист на Rust с красивым интерфейсом на `ratatui`.

## Что умеет

- двухпанельный TUI-интерфейс
- фильтры `All / Active / Done`
- добавление, редактирование, удаление задач
- удаление всех выполненных задач
- полная очистка локальных данных из интерфейса
- автоматическое сохранение в `~/.todo_rust/tasks.json`

## Требования

Нужен установленный Rust toolchain:

```bash
curl https://sh.rustup.rs -sSf | sh
source "$HOME/.cargo/env"
```

Если Rust уже установлен, этот шаг пропусти.

## Как собрать

Перейди в папку проекта:

```bash
cd /Users/antonaleynichenko/Documents/todo_rust
```

Собери проект:

```bash
cargo build
```

Сборка в release-режиме:

```bash
cargo build --release
```

## Как запустить

Из папки проекта:

```bash
cargo run
```

Или release-версию:

```bash
./target/release/todo_rust
```

Если команда `cargo` не найдена:

```bash
source "$HOME/.cargo/env"
```

## Управление в TUI

- `↑ / ↓` или `j / k` — перемещение по задачам
- `Enter` или `Space` — отметить выполненной / вернуть в активные
- `a` — добавить задачу
- `e` — переименовать выбранную задачу
- `d` — удалить выбранную задачу
- `x` — удалить все выполненные задачи
- `Shift+X` — удалить все локальные данные и файл `~/.todo_rust/tasks.json`
- `1 / 2 / 3` — фильтр `All / Active / Done`
- `Tab` или `/` — переключить фильтр
- `Esc` — закрыть окно ввода или отменить опасное действие
- `q` — выход

## Где лежат данные

Задачи сохраняются в:

```bash
/Users/antonaleynichenko/.todo_rust/tasks.json
```

Папка `.todo_rust` скрытая, поэтому в файловом менеджере или VS Code её не видно, пока не включён показ скрытых файлов.

## Как удалить приложение и его данные

1. Удали все задачи прямо из TUI через `Shift+X`, затем подтверди удаление клавишей `y`.
2. Удали папку проекта:

```bash
rm -rf /Users/<user>/Documents/todo_rust
```

Если хочешь удалить только скомпилированные файлы, достаточно удалить папку `target/` внутри проекта.
