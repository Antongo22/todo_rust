# todo_rust

Красивый консольный todo-лист на Rust с интерфейсом на `ratatui`.

## Возможности

- аккуратный двухпанельный интерфейс
- фильтры `All / Active / Done`
- добавление, редактирование, удаление и переключение статуса задач
- автоматическое сохранение в `~/.todo_rust/tasks.json`
- удобное управление только с клавиатуры

## Управление

- `↑ / ↓` или `j / k` — перемещение по задачам
- `Enter` или `Space` — отметить выполненной / вернуть в активные
- `a` — новая задача
- `e` — переименовать выбранную
- `d` — удалить выбранную
- `x` — удалить все выполненные
- `1 / 2 / 3` — фильтр `All / Active / Done`
- `Tab` или `/` — переключить фильтр
- `Esc` — закрыть окно ввода
- `q` — выход

## Запуск

```bash
cargo run
```

Если Rust не установлен:

```bash
curl https://sh.rustup.rs -sSf | sh
source "$HOME/.cargo/env"
```
